# Granular

Experiment testing out the GranularPlayer class in DaisySP. Uses a triggered 2 second sample recorder.

## Controls

| Control | Description | Comment |
| --- | --- | --- |
| Ctrl 1 | Size | Grain size 1ms to 200ms |
| Ctrl 2 | Speed | Speed of grain playback -2x to +2x |
| Ctrl 3 | Pitch | Pitch (transposition) of the grains, -1 to +1 octave |
| Ctrl 4 | Mix | Controls the Attack/Decay time of both the modulator and carrier |
| Ctrl 5 | Level | Controls the sustain level (volume while note is held) |
| Ctrl 6 |  |  |
| 3-Way Switch 1 |  | |
| 3-Way Switch 2 |  |  |
| 3-Way Switch 3 |  |  |
| Dip Switch 1 |  |  |
| Dip Switch 2 |  |  |
| FS 1 | Bypass |  |
| FS 2 | Trigger | Trigger recording a sample (will repeat until another sample is recorded) |
| LED 1 | Bypass Indicator |  |
| LED 2 | Recording Indicator | |
| Audio In 1 | N/A | Ignores Right channel input  |
| Audio Out 1 | Stereo | |

Expression controls Knob 2 only, Speed. Plug in expression pedal to use, move knob 2 to deactivate expression control.



## Build

Granular runs in Flash memory on the Daisy Seed.
